# Twitterコンテンツ作成ガイドライン

このガイドラインは、長文記事をTwitter（X）配信に適した形式に変換するためのルールです。

---

## 1. 基本方針

### 目的
- 記事の核心部分を凝縮し、Twitterユーザーの興味を引く
- 記事本体への誘導（クリック率向上）
- フォロワーとのエンゲージメント促進

### ターゲット
- Twitterでアウトドア・キャンプ・登山情報を探しているユーザー
- 購買検討中のユーザー（「どれがいい？」と迷っている人）
- 隙間時間にサクッと情報を得たい人

---

## 2. コンテンツ形式

### 1話完結シリーズ形式（採用）

1つの記事から**5〜8個の独立した投稿**を作成する。

#### 重要な考え方
- **各投稿は単体で完結**する（スレッドのようにつなげて読む必要なし）
- タイムラインで1つだけ見ても**価値がある**内容にする
- 同じ記事から派生していても、**順番は関係ない**
- 各投稿が異なる切り口・トピックを扱う

#### 1話完結のメリット
- ユーザーはどの投稿から見ても理解できる
- リツイートされやすい（単体で意味が通じるから）
- 投稿タイミングを分散できる（毎日1つずつなど）
- 反応が良かった切り口を分析しやすい

---

## 3. ツイート作成ルール

### 文字数
- **1ツイート：100〜140文字**を目安（日本語）
- 余白を残してリツイートしやすくする

### フック（1ツイート目）の書き方

#### ✅ 効果的なフック例
- **問いかけ型**: 「スノーピークのペグ、4サイズあるけど結局どれがいいの？」
- **衝撃事実型**: 「実は30cmが最強説」
- **失敗談型**: 「付属ペグで痛い目見た人、正直に手を挙げて✋」
- **数字型**: 「15年使っても壊れないハンマーの話」

#### ❌ NGなフック
- 「〜について解説します」（退屈）
- 記事タイトルをそのまま貼る（長すぎ）
- いきなりスペック羅列（興味が湧かない）

### 本編ツイートの書き方

#### ポイント
1. **1ツイート1メッセージ**に絞る
2. **具体的な数字**を入れる（「軽い」→「75g」）
3. **比較・対比**を使う（「Pro.Sは5,280円、Pro.Cは8,470円」）
4. **結論ファースト**で書く
5. **箇条書き・絵文字**で視認性UP

#### 変換例

**❌ 記事そのまま（NG）**
```
ソリッドステーク30は最も汎用性が高いサイズで、テントにもタープにも対応できて、
硬い地面から普通の地面までカバーできます。初心者の方にも扱いやすい長さです。
```

**✅ Twitter用に変換（OK）**
```
✅ 迷ったら30cm一択

・テントにもタープにも使える
・硬い地面〜普通の地面までOK
・1本495円でコスパ◎

まず10本揃えれば大体なんとかなる
```

### CTAツイートの書き方

#### 効果的なCTA例
- 「詳しい選び方は記事で👇」
- 「スペック比較表は記事にまとめた→」
- 「あなたはどっち派？リプで教えて👇」

## 4. 記事→Twitterの変換手順

### Step 1: トピックを抽出
記事から**5〜8個の独立したトピック**を抜き出す。（/summary/配下の同名ファイルのサマリファイルも参照）

#### トピック抽出の視点
| 視点 | 例 |
|------|-----|
| 結論・選び方 | 「迷ったら30cmを選べ」 |
| サイズ別比較 | 「20cm vs 30cm vs 40cm」 |
| 製品比較 | 「Pro.S vs Pro.C どっち？」 |
| 意外な事実 | 「曲がっても叩けば直せる」 |
| 失敗談・注意点 | 「付属ペグで痛い目見た話」 |
| 使い方・コツ | 「抜けないペグの抜き方」 |
| コスパ・予算 | 「最初に揃えるべきセット」 |
| ライバル比較 | 「ソリステ vs エリステ」 |

### Step 2: 各トピックを1投稿に仕上げる

各投稿は以下の構造で**単体完結**させる：

```
【フック】1〜2行で興味を引く
　↓
【本編】要点を3〜5行で伝える
　↓
【締め・CTA】記事リンク or 質問で締める
```

### Step 3: 独立性チェック
各投稿について確認：
- [ ] この投稿だけ見て意味が通じるか？
- [ ] 「前のツイート見て」と言わずに済むか？
- [ ] リツイートされても恥ずかしくないか？

### Step 4: 元記事への誘導文言を追加
各投稿の最後に、元記事への誘導文言とURLを追加する。

**記事URLの形式**：
`https://sensei-omoi.flow-t.net/[記事ファイル名]#section-[番号]`

**ページ内リンク（アンカー）の決定方法**：
- サイトではh2, h3見出しに対して`section-1`, `section-2`...と連番でidが付与される
- 投稿内容に最も関連するセクションのidを特定して誘導先に指定
- 元記事の見出し構造を確認し、該当するセクション番号を数えて決定

**例**：
```
→ 各アイテムの選び方と予算感
https://sensei-omoi.flow-t.net/beginner_hiking_start_guide#section-6
```

**重要：定型文での誘導はNG**
「詳しくはこちら→」のような定型文はスパム感があり逆効果。
各投稿の内容を読んで興味を持った人が「もっと知りたい」と思う具体的な内容を提示すること。

**良い誘導文の作り方**：
1. その投稿で触れたトピックを深掘りする内容を提示
2. 投稿では語りきれなかった具体例やリストを予告
3. 読者が「続きが気になる」と思える問いかけ

**例**：
- ❌ 悪い例：「詳しくはこちら→」（定型文）
- ✅ 良い例：
```
→ 各アイテムの選び方と予算感
https://sensei-omoi.flow-t.net/beginner_hiking_start_guide
```
- ✅ 良い例：
```
→ 失敗しないための持ち物チェックリスト
https://sensei-omoi.flow-t.net/beginner_hiking_start_guide
```

**注意**：
- ハッシュタグは使用しない（スパム感があり、エンゲージメント低下につながるため廃止）
- 誘導文言は各投稿に必ず1つ入れる
- 誘導文とURLは改行して記載する

### Step 5: 画像生成プロンプトを追加
各投稿に添付する画像を生成するためのプロンプトを追加する。

**プロンプトの書き方**：
1. 投稿内容を視覚的に補強する画像を指定
2. 具体的な構図・被写体・背景を明記
3. 比較系の投稿は「左右比較の構図」を活用

**効果的なプロンプト例**：
- 製品紹介：「ソリッドステーク4サイズを長さ順に並べたフラットレイ。木目調の背景。」
- 比較系：「左右比較の構図。左側：大股で歩いて疲れた表情の登山者。右側：小股でリラックスして歩く登山者。」
- 警告系：「濡れた綿Tシャツが肌に張り付いて寒そうに震える登山者。危険を警告する雰囲気。」
- ハウツー系：「足裏全体が岩場にフラットに着地している瞬間のクローズアップ。」

**記載形式**：
```
> **Image Prompt:** [プロンプト内容]
```

## 5. ファイル形式

### 保存場所
`/twitter/[記事ファイル名と同名].md`

### ファイル構成

```markdown
# [記事タイトル] - Twitterコンテンツ

元記事: /article/[ファイル名].md
作成日: YYYY-MM-DD
投稿数: 6

---

## 投稿一覧

### 投稿1：[切り口]（例：結論・選び方）
[本文]

---

### 投稿2：[切り口]
[本文]

---

### 投稿3：[切り口]
[本文]

...

```

### 命名規則
- 各投稿には「投稿1」「投稿2」のように番号を振る
- 番号の後に「切り口」を記載する（投稿順序は任意）

## 変換例サンプル

### 元記事: スノーピーク ペグ・ハンマー完全ガイド

以下は同じ記事から作成した**独立した6つの投稿**。どれから読んでもOK。

---

**投稿1：結論・選び方**
```
スノーピークのペグ、4サイズあるけど
迷ったら30cm一択

✅ テントにもタープにも使える
✅ 硬い地面でも普通の地面でもOK
✅ 1本495円でコスパ最強

まず10本買っとけば間違いない

→ サイズ別の使い分けとセット例はプロフから
```

---

**投稿2：サイズ別解説**
```
ソリステのサイズ選び、これで完璧👇

・20cm（75g）→ ソロ用、軽量化したい人
・30cm（180g）→ 万能、迷ったらコレ
・40cm（330g）→ 大型タープのメイン用
・50cm（540g）→ 砂地専用

50cmは1本540gでマジ重いから注意な

→ ペグの本数と予算の目安はプロフから
```

---

**投稿3：ハンマー比較**
```
スノーピークのハンマー
Pro.SとPro.C、どっちがいい？

【Pro.S】5,280円
→ スチール製、頑丈、初心者向け

【Pro.C】8,470円
→ 銅ヘッド、衝撃吸収、打ち心地◎

結論：最初はPro.Sで十分
こだわりたくなったらPro.Cに昇格

→ 打ち心地の違いとレビューはプロフから
```

---

**投稿4：意外な事実**
```
スノーピークのペグ、曲がっても捨てないで

実は硬い面に置いてハンマーで叩けば
真っ直ぐに直せる（公式推奨）

「消耗品」じゃなくて「一生もの」

これ知らなくて捨てた人いたら
ドンマイとしか言えない

→ 直し方の詳細とメンテナンス方法はプロフから
```

---

**投稿5：失敗談・注意点**
```
テント付属のペグで痛い目見た人へ

河原の石混じりの地面→曲がる
強風の夜→抜ける
最悪→テント飛ばされる

ソリステなら石に当たっても曲がらない
「ペグは消耗品」を覆した製品

安心を買うと思えば高くない

→ 平地用ペグからあなたが変えるべき理由はプロフから
```

---

**投稿6：コスパ・予算**
```
ソリステ揃えるといくらかかる？

【最小構成】約8,300円
・30cm × 6本（3,000円）
・Pro.S（5,280円）

【標準構成】約11,800円
・30cm × 10本
・20cm × 4本（予備）
・Pro.S

一生使えると思えば安い投資

→ 用途別のおすすめセットと同時購入リストはプロフから
```

---

### ポイント解説

| 投稿 | 切り口 | 独立性 |
|------|--------|--------|
| 1 | 結論ファースト | ✅ これだけで選べる |
| 2 | サイズ別整理 | ✅ 比較だけで完結 |
| 3 | 製品比較 | ✅ ハンマーの話だけ |
| 4 | 意外な豆知識 | ✅ 小ネタとして成立 |
| 5 | 失敗談 | ✅ 共感で拡散狙い |
| 6 | 予算感 | ✅ 金額だけで完結 |

どの投稿も「前のを見て」と言わずに成立している。

# インスタリール台本作成ルール

## 絶対にやってはいけないこと

### ❌ スペック・情報の羅列
```
悪い例:
「カマボコテントは収容人数5名、重量19.5kg、価格75,900円です」
```
→ 誰も興味ない。カタログ読めばいい。

### ❌ 淡々とした説明
```
悪い例:
「次にワンポールテントを紹介します。特徴は設営が簡単なことです」
```
→ 寝る。学校の授業じゃない。

### ❌ 最初からポジティブ
```
悪い例:
「DODのテント、めっちゃおすすめです！」
```
→ 広告臭い。信用されない。

### ❌ 「おすすめです！」で終わる
→ だから何？行動につながらない。

---

## 必ず入れる要素

### ✅ 3秒で刺すフック
最初の3秒で離脱される。ネガティブ or 意外性で引き込む。

```
良い例:
「DODテント買って後悔してる」
「これ買うな、マジで」
「キャンプ場で恥かいた話していい？」
```

### ✅ 共感できる失敗談・あるある
視聴者が「わかる〜」と思える体験談を入れる。

```
良い例:
「20kgで腰やった」
「設営で2時間かかって日が暮れた」
「雨降ってきて泣きそうになった」
```

### ✅ 感情の起伏（転換）
ネガティブ → ポジティブ、または逆。平坦はダメ。

```
良い例:
「後悔してる……って言いたかったけど、正直めっちゃ気に入ってる」
「でも中に入った瞬間『うわ、広っ！！』」
```

### ✅ 具体的な数字
抽象的な表現は刺さらない。

```
悪い例: 「けっこう重い」「広い」
良い例: 「20kg」「8畳」「設営30分」「3回聞かれた」
```

### ✅ 本音・毒舌
きれいごとだけじゃ信用されない。

```
良い例:
「いきなりカマボコは挫折する」
「体力に自信ないなら買うな」
「これ持てる人だけ買って」
```

### ✅ 参加型CTA
コメント誘導でアルゴリズム優遇される。

```
良い例:
「あなたはどれ派？コメントで教えて」
「迷ったらコメントして！」
「どっち買うか投票して」
```

---

## 構成パターン

### パターン1: 失敗談→転換（推奨）
1. ネガティブフック（後悔した/失敗した）
2. 具体的な失敗エピソード
3. 「でも実は…」で転換
4. 本当の価値を伝える
5. 他の選択肢も提示
6. 参加型CTA

### パターン2: 対決・比較
1. 「どれがいい？」でフック
2. 製品Aの特徴（1文+判定）
3. 製品Bの特徴（1文+判定）
4. 製品Cの特徴（1文+判定）
5. 初心者向けの結論
6. 「あなたはどれ派？」

### パターン3: クイズ形式
1. 「何問正解できる？」でフック
2. Q1（意外な事実）
3. Q2（知っておくべき数字）
4. Q3（選ぶべき正解）
5. 結果発表で煽る
6. フォロー誘導

| プラットフォーム | 推奨尺 |
|----------------|-------|
| リール | 30〜60秒 |
| TikTok | 15〜60秒 |
| ショート | 30〜60秒 |

## 台本フォーマット

```markdown
**【0:00-0:05】セクション名**

📱 テキスト: 「画面に出すテキスト」

🎤 ナレーション:
> 「話す内容」

🎬 映像: 映像のイメージ

💡 ポイント: 編集時の注意点
```

## 参考：ダメな例 vs 良い例

### ダメな台本
```
「DODテントを紹介します。
カマボコテントは収容人数5名、重量19.5kg、価格75,900円。
次にワンポールテント。設営が簡単で軽量です。
タケノコテントは8畳の広さがあります。
以上、DODテントの紹介でした。おすすめです！」
```
→ 誰も見ない

### 良い台本
```
「DODテント買って後悔してる。
カマボコテント、おしゃれだと思って買ったら20kgあった。
車から降ろすだけで汗だく、設営で腰おわった。
……でもね、中入った瞬間『うわ、広っ！！』
正直今は自慢のテント。
初心者はワンタッチから始めて。いきなりカマボコは挫折する。
あなたはどれ派？コメントで教えて！」
```
→ 最後まで見る

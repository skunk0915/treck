# インスタリール台本作成ルール
指定された記事を元にInstagramのユーザー特性やリール動画の特性を考慮したコンテンツを作成する。
（/summary/配下の同名ファイルのサマリファイルも参照）

## 絶対にやってはいけないこと

### ❌ スペック・情報の羅列
```
悪い例:
「カマボコテントは収容人数5名、重量19.5kg、価格75,900円です」
```
→ 誰も興味ない。カタログ読めばいい。

### ❌ 淡々とした説明
```
悪い例:
「次にワンポールテントを紹介します。特徴は設営が簡単なことです」
```
→ 寝る。学校の授業じゃない。

### ❌ 最初からポジティブ
```
悪い例:
「DODのテント、めっちゃおすすめです！」
```
→ 広告臭い。信用されない。

### ❌ 「おすすめです！」で終わる
→ だから何？行動につながらない。

## 必ず入れる要素

### ✅ 3秒で刺すフック
最初の3秒で離脱される。ネガティブ or 意外性で引き込む。

```
良い例:
「DODテント買って後悔してる」
「これ買うな、マジで」
「キャンプ場で恥かいた話していい？」
```

### ✅ 共感できる失敗談・あるある
視聴者が「わかる〜」と思える体験談を入れる。

```
良い例:
「20kgで腰やった」
「設営で2時間かかって日が暮れた」
「雨降ってきて泣きそうになった」
```

### ✅ 感情の起伏（転換）
ネガティブ → ポジティブ、または逆。平坦はダメ。

```
良い例:
「後悔してる……って言いたかったけど、正直めっちゃ気に入ってる」
「でも中に入った瞬間『うわ、広っ！！』」
```

### ✅ 具体的な数字
抽象的な表現は刺さらない。

```
悪い例: 「けっこう重い」「広い」
良い例: 「20kg」「8畳」「設営30分」「3回聞かれた」
```

### ✅ 本音・毒舌
きれいごとだけじゃ信用されない。

```
良い例:
「いきなりカマボコは挫折する」
「体力に自信ないなら買うな」
「これ持てる人だけ買って」
```

### ✅ 参加型CTA
コメント誘導でアルゴリズム優遇される。

```
良い例:
「あなたはどれ派？コメントで教えて」
「迷ったらコメントして！」
「どっち買うか投票して」
```

## 構成の考え方

> **重要**: 以下のパターンはあくまで参考例。コンテンツの内容・特性に合わせて都度最適な構成を考えること。定型パターンに当てはめるのではなく、「この内容を伝えるにはどういう流れが最も刺さるか？」を考える。

### 構成設計の手順
1. 記事の核となるメッセージを1つ決める
2. そのメッセージが最も刺さる視聴者の感情を考える（不安？好奇心？共感？）
3. その感情を引き出すフックを考える
4. フックから結論までの自然な流れを設計する
5. 参加型CTAで終わる

### 1記事から複数コンテンツを作る
情報量の多い記事は、1つのコンテンツに詰め込まず、トピックごとに分割して複数のリールを作成する。

**例**: 「登山の始め方ガイド」の場合
- コンテンツ1: 三種の神器とは？
- コンテンツ2: 絶対着ちゃダメな服
- コンテンツ3: 最初に登るべき山
- コンテンツ4: 初心者がやりがちな失敗

それぞれ独立したコンテンツとして成立させる。同じ記事でもトピックが変われば最適な構成も変わる。

### 参考パターン（必要に応じて使用・アレンジ）

**失敗談→転換**
ネガティブから入ってポジティブで終わる。共感を得やすい。

**対決・比較**
複数の選択肢がある場合に有効。判断基準を示せる。

**クイズ形式**
知識系コンテンツに有効。視聴者を参加させられる。

**ストーリーテリング**
時系列で体験を語る。感情移入しやすい。

**逆説・意外性**
「実は◯◯だった」で常識を覆す。注目を集めやすい。

**ステップ解説**
ハウツー系に有効。保存されやすい。

| プラットフォーム | 推奨尺 |
|----------------|-------|
| リール | 30〜60秒 |
| TikTok | 15〜60秒 |
| ショート | 30〜60秒 |

## 台本フォーマット

```markdown
**【0:00-0:05】セクション名**

📱 テキスト: 「画面に出すテキスト」

🎤 ナレーション:
> 「話す内容」

🎬 映像: 映像のイメージ

💡 ポイント: 編集時の注意点
```

## 参考：ダメな例 vs 良い例

### ダメな台本
```
「DODテントを紹介します。
カマボコテントは収容人数5名、重量19.5kg、価格75,900円。
次にワンポールテント。設営が簡単で軽量です。
タケノコテントは8畳の広さがあります。
以上、DODテントの紹介でした。おすすめです！」
```
→ 誰も見ない

### 良い台本
```
「DODテント買って後悔してる。
カマボコテント、おしゃれだと思って買ったら20kgあった。
車から降ろすだけで汗だく、設営で腰おわった。
……でもね、中入った瞬間『うわ、広っ！！』
正直今は自慢のテント。
初心者はワンタッチから始めて。いきなりカマボコは挫折する。
あなたはどれ派？コメントで教えて！」
```
→ 最後まで見る

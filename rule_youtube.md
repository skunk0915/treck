# YouTube台本作成ルール

指定された記事を元にYouTubeショート動画用の台本を作成する。
（/article/配下の記事と/summary/配下の同名ファイルのサマリファイルを参照）

---

## 1. 基本方針

### キャラクター設定
**character_sheet.mdを必ず参照すること。**

- **JK**：登山初心者の女子高生。好奇心旺盛、見た目重視。素朴な疑問や不満を正直に口にする。
  - 語尾：「〜ですよね」「〜かも」「え〜」
  - 一人称：「私」
  - 相手：「先生」
  
- **先生**：登山歴20年のベテランガイド。無骨で質実剛健、落ち着いた「師匠」のような雰囲気。
  - 語尾：「〜だ」「〜だな」「〜ぞ」「〜だろう」
  - 一人称：「私」
  - 笑い声は「ふむ…」「はは」と控えめ。感嘆符（！）は少なめ。

### 動画形式
- 2人の掛け合い形式で進行
- 1つの記事から1〜複数の動画を作成可能
- 各動画は独立して視聴できる内容にする

---

## 2. ファイル形式

### 保存場所
`/youtube/[記事ファイル名と同名].csv`

### CSV形式
```
話者名,セリフ
JK,セリフ内容
先生,セリフ内容
```

### 表記ルール
- 1行につき1セリフ
- 話者名は「JK」または「先生」のみ
- セリフはダブルクォーテーションで囲わない（CSVの標準形式に従う）
- セリフ内にカンマが含まれる場合は、セリフ全体をダブルクォーテーションで囲む

---

## 3. 最重要ルール：記事の会話をそのまま活かす

### ❌ やってはいけないこと
- 記事の会話を要約・簡略化する
- オリジナルの会話を勝手に作る
- キャラクターの口調を変える

### ✅ やるべきこと
- **記事の会話をそのまま使う**
- 50文字以内の自然な位置で区切るだけ
- キャラクターシートの口調を厳守

---

## 4. セリフの区切り方

### 文字数制限
- **1セリフ：50文字以下**
- 字幕として画面内に収めるため
- 日本語として自然な位置で区切る

### 複数行にまたがってOK
同じ話者のセリフが長い場合、複数行に分けて書く。

**例：記事の会話**
```
先生: キャンプスタイルや人数、使用シーンによって最適なタープは変わるからだ。
```

**変換後**
```csv
先生,キャンプスタイルや人数、使用シーンによって
先生,最適なタープは変わるからだ
```

### 区切りポイント
- 句点（。）
- 読点（、）の後
- 接続詞の前（「でも」「だから」「そして」など）
- 強調したい言葉の前後

---

## 5. リスト・表の話し言葉変換

### 重要ルール
記事中のリストや表は**画像として動画内に別途表示**する。
セリフではリストを読み上げず、**画像を見せる前提の言い回し**に変換する。

### 変換パターン

| 記事の内容 | 話し言葉への変換 |
|-----------|-----------------|
| スペック表 | 「スペックを画面に出すぞ」「スペックを見てみよう」 |
| 比較表 | 「比較表を画面に出すぞ」「比較を画面で見てくれ」 |
| 注意点リスト | 「注意点を画面で確認してくれ」 |
| メリット・デメリット | 「それぞれの特徴を画面に出すぞ」 |
| 手順リスト | 「手順を画面で見てくれ」 |
| まとめリスト | 「ポイントを画面に出すぞ」 |

### 変換例

**❌ NG（リストを読み上げ）**
```
先生,必要なものは登山靴とザックとレインウェアの3つだ
```

**✅ OK（画像参照を前提）**
```csv
先生,必要なものは3つある
先生,画面に出すぞ
JK,あ、これが三種の神器ってやつですか？
先生,そうだ
```

**✅ OK（表を見せる場合）**
```csv
先生,スペックを見てみよう
JK,うわ、ほんとに大きいですよね
先生,カマボコテントと連結できる
```

---

## 6. 会話の流れ

### 基本は記事の構成に従う
記事の見出し・セクション順に会話を進める。

### 会話のテンポ
- JKと先生が交互に話す（記事通り）
- 記事にないオリジナルの掛け合いは追加しない

---

## 7. チェックリスト

台本完成後、以下を確認：

- [ ] 全セリフが50文字以下か？
- [ ] 話者名は「JK」または「先生」のみか？
- [ ] 記事の会話をそのまま使っているか？（要約していないか？）
- [ ] キャラクターシートの口調を守っているか？
  - 先生：「〜だ」「〜だな」「〜ぞ」（丁寧語NG）
  - JK：「〜ですよね」「〜かも」（「〜じゃん」等カジュアルすぎNG）
- [ ] リスト・表は画像表示を前提にした言い回しになっているか？

## 8. サンプル台本

### 記事：DOD タープ完全ガイド（冒頭部分）

```csv
話者名,セリフ
JK,先生！DODのタープ見てたんですけど
JK,「いつかのタープ」とか「チーズタープ」とか
JK,名前が意味不明すぎません？
JK,しかも種類多すぎ！
先生,ふむ、DODの製品名は確かに独特だな
先生,だが、種類が多いのには理由がある
先生,キャンプスタイルや人数、使用シーンによって
先生,最適なタープは変わるからだ
JK,えー、でもタープって
JK,日除けになればいいんじゃないんですか？
先生,そう単純ではない
先生,例えば、ソロキャンプで使うのか
先生,ファミリーで使うのか
先生,焚き火を楽しみたいのか
先生,雨対策を重視するのか
先生,それによって選ぶべきタープは全く違う
```

### ポイント
- 記事の会話をそのまま使用
- 50文字以内で自然に区切り
- 先生は「〜だ」「〜だな」、JKは「〜ですか」「〜ですよね」

## 9. 1記事から複数動画を作る場合

情報量の多い記事は、トピックごとに分割して複数の動画を作成する。

### 例：登山の始め方ガイド
- 動画1: 三種の神器って何？
- 動画2: 絶対着ちゃダメな服
- 動画3: 初心者におすすめの山
- 動画4: よくある失敗

各動画は独立したCSVファイルとして作成し、ファイル名に連番を付ける。
例：`beginner_hiking_start_guide_01.csv`, `beginner_hiking_start_guide_02.csv`

# DOD テーブル完全ガイド サマリ

## 1. DODテーブルの概要

DOD（ディーオーディー）は、ユニークなネーミングとデザイン性の高さで人気のアウトドアブランド。テーブル製品は機能性とリーズナブルな価格帯で、ソロキャンプからファミリーキャンプまで幅広いニーズに対応している。

### 主なテーブルシリーズ
- **収納重視系**: グッドラックテーブル、マルチキッチンテーブル
- **焚き火対応系**: テキーラテーブルシリーズ
- **ソロ・調理特化系**: ソトメシンガーZ/RX、ダイキチテーブル
- **天然木デザイン系**: キャナリーテーブル、ジミニーテーブル
- **特殊用途系**: ワンポールテントテーブル、タナカテーブル

## 3. 素材別特性と注意点

### MDF素材（グッドラックテーブル、マルチキッチンテーブル等）
- 軽量で加工しやすい
- **注意**: カビや熱に弱い、焚き火近くでの使用禁止
- 水滴跡が残りやすい場合あり

### 天然木・ブナ（キャナリーテーブル、ジミニーテーブル等）
- 高級感、温かみのあるデザイン
- 使い込むほど風合いが増す
- **注意**: 直射日光、高湿度、カビに注意
- 個体差あり（木目、色合い、反り）

### スチール・鉄（テキーラテーブル、ソトメシンガー等）
- 耐熱性に優れる
- 焚き火周りでの使用に最適
- やや重量がある

### アルミ合金（タナカテーブル等）
- 軽量で持ち運びやすい
- 錆びにくい

## 4. 天然木テーブルのメンテナンス

### 日々のお手入れ
- 柔らかい布で乾拭き
- 汚れは固く絞った布で水拭き後、すぐ乾拭き
- 化学雑巾、揮発性溶剤は使用しない

### オイル仕上げのテーブル
- 定期的に家具用オイルを塗布（年1〜2回推奨）
- 傷やシミはサンドペーパー（#400程度）で研磨後オイル塗布

### カビ対策
- 換気と除湿を徹底
- 使用後は汚れを落とし完全乾燥してから保管
- 高温多湿、直射日光を避ける
- カビ発生時は中性洗剤を薄めた水で拭き取り、エタノールも有効（色落ちに注意）
- 塩素系漂白剤は使用禁止

**参考情報**: 「使用後は汚れを落とし、完全に乾燥させてから保管することが指示されています。濡れたまま保管すると、カビや腐食、劣化の原因となります」<details><summary>出典</summary>`https://dod.camp`</details>

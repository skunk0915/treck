# 登山パッキング 調査サマリ

## 1. パッキングの基本原則

### 1.1 重心配置の基本
- **最重要**: 重い荷物をバックパックの**背中側**、かつ**上部**に寄せて配置する
- **理由**: バックパックの重心が体の重心に近づき、体への負担が軽減され、バランスが取りやすくなる
- **具体的な位置**: 肩甲骨のあたりに重いものがくるようにすると良い
- **参考情報**: 「重い荷物をバックパックの背中側、かつ上部に寄せて配置することにより、バックパックの重心が体の重心に近づき、体への負担が軽減され、バランスが取りやすくなります。」<details><summary>出典</summary>`https://yamap.com/magazine/70369`</details>

**クライミング時の例外**:
- クライミングなど体を大きく動かす場合は、重心を背中の**中央**あたりに置くのが理想
- **参考情報**: 「クライミングなどの場合は、重心を背中の中央あたりに置くのが理想とされています。」<details><summary>出典</summary>`https://mammut.jp/magazine/`</details>

### 1.2 ゾーニング（区分け）配置
バックパックを上・中・下、背中側・外側などのゾーンに分けて考える：

| ゾーン | 入れるもの | 具体例 |
|--------|------------|--------|
| 下部（最下部） | あまり使わない軽いもの | 寝袋、着替え、厚手の防寒着 |
| 背中側（上部〜中央） | 重いもの | 水、食料、燃料、クッカー、テント本体 |
| 外側 | 軽めで下部よりは取り出す機会があるもの | フリース、ジャケット |
| 上部・雨蓋 | よく使うもの、緊急時に必要なもの | エマージェンシーキット、行動食、ヘッドライト、レインウェア |
| サイドポケット | 頻繁に取り出すもの | 水筒、日焼け止め、虫除けスプレー |

- **参考情報**: 「使用頻度の高いものは上部や取り出しやすい場所に、使用頻度の低いものは底部に収納します。」<details><summary>出典</summary>`https://yamahack.com/5844`</details>

### 1.3 左右のバランス
- 荷物を詰める際は左右の重さが偏らないように注意
- 片側に重さが集中すると筋肉に不自然な負担がかかり、痛みや怪我の原因になる
- **参考情報**: 「荷物を詰める際は、左右の重さが偏らないように注意しましょう。」<details><summary>出典</summary>`https://yamakei-online.com/`</details>

## 2. パッキングテクニック

### 2.1 隙間なく詰める
- ザックの中に隙間があると、歩行中に荷物が動いてバランスを崩しやすくなる
- スタッフバッグなどを活用して荷物を小分けにし、隙間なく均等に詰める
- 寝袋はきれいにたたむのではなく、足元から押し込むようにするとコンパクトに収まる
- **参考情報**: 「隙間なく詰めることで、荷物の揺れを防ぎ、安定した歩行をサポートします。」<details><summary>出典</summary>`https://alpen-group.jp/`</details>

### 2.2 縦に詰めることを意識
- ザックは基本的に縦長なので、荷物も縦向きに入れることで取り出しやすくなる
- スペースを効率的に活用できる
- **参考情報**: 「荷物を縦向きに入れることを意識すると、取り出しやすく、スペースを有効活用できます。」<details><summary>出典</summary>`https://mt-inn.jp/`</details>

### 2.3 パッケージの事前処理
- 長期間の山行では、食品のパッケージがゴミになるため、自宅で中身を密閉袋に移し替えておくと良い
- ゴミの量を減らし、軽量化にも貢献

## 3. スタッフバッグの活用

### 3.1 スタッフバッグの種類
| 種類 | 特徴 | 用途 |
|------|------|------|
| 非防水タイプ（巾着型・ジッパー型） | 物の出し入れがしやすく、安価 | すぐに取り出したい小物、濡れても問題ないもの |
| 防水タイプ（ドライバッグ） | ロールトップ式、シームテープ処理で高い防水性 | ダウン製品、着替えなど絶対に濡らしたくないもの |

### 3.2 容量の目安
| 容量 | 用途 |
|------|------|
| 1〜2L | 小物類（電子機器、薬など） |
| 5〜7L | 着替え |
| 10L以上 | 寝袋などの大物 |

- **参考情報**: 「スタッフバッグには大きく分けて、非防水の巾着型・ジッパー型と、完全防水のドライバッグがあります。」<details><summary>出典</summary>`https://yamatabitabi.com/archives/209879/`</details>

### 3.3 活用のコツ
- 色違いのスタッフバッグを使うと、荷物の分類がしやすくなる
- カテゴリ別にまとめることで必要なものを素早く取り出せる
- **参考情報**: 「色違いのスタッフバッグを使うと、さらに荷物の分類がしやすくなります。」<details><summary>出典</summary>`https://yamap.com/`</details>

## 4. 防水対策

### 4.1 複数の防水レイヤー
| 方法 | 説明 | 長所・短所 |
|------|------|------------|
| レインカバー | リュックの外側を覆う | 最も一般的だが、完全防水ではない。強い雨では隙間から浸水することも |
| パックライナー | リュックの内側に入れる大型の防水袋 | 荷物全体を雨から守るのに非常に効果的 |
| 防水スタッフバッグ | 個別の荷物を守る | 着替えやダウン製品など重要なものを個別に保護 |
| ジッパー付き密閉袋 | 小物の防水対策 | コスト安く手軽 |

- **ベストプラクティス**: パックライナーとザックカバーを併用することで、ザック本体が濡れるのを防ぎつつ、中身をしっかりと保護できる
- **参考情報**: 「パックライナーとザックカバーを併用することで、ザック本体が濡れるのを防ぎつつ、中身をしっかりと保護できます。」<details><summary>出典</summary>`https://yamahack.com/`</details>

## 5. 外付け・圧縮バンド（コンプレッションストラップ）

### 5.1 コンプレッションストラップの役割
- **荷物の圧縮と安定**: 荷物が少ない時にストラップを締めることでザックの体積を圧縮し、中の荷物が動くのを防ぐ
- **体へのフィット感向上**: 常にザックを背中に引き寄せるように締めることが重要
- **外付け荷物の固定**: ストラップを伸ばすことで、スリーピングマットなどをザックの外側に固定できる
- **参考情報**: 「荷物が少ない時にストラップを締めることで、ザックの体積を圧縮し、中の荷物が動くのを防ぎます。」<details><summary>出典</summary>`https://yamahack.com/`</details>

### 5.2 外付けの注意点
- 外付けする荷物が横に張り出しすぎると、狭い登山道で木の枝に引っかかったり、他の登山者にぶつかったりする
- **原則**: できる限りザックの中に収納することが推奨
- **参考情報**: 「できる限りザックの中に収納することが推奨されています。」<details><summary>出典</summary>`https://yamahack.com/`</details>

## 6. ザックのフィッティングと背負い方

### 6.1 調整の順序
1. **ストラップをすべて緩める**: ショルダーハーネス、ヒップベルト、チェストストラップ、スタビライザーをすべて最大限に緩める
2. **荷物を詰める**: 10kg程度の荷物を入れてフィッティングすると実際の使用に近い状態で確認できる
3. **ザックを背負う**: 片膝に乗せるなどして背中に担ぎ上げる
4. **腰ベルト（ヒップベルト）を締める**: 最初に調整
5. **ショルダーハーネスを調整**: 腰ベルト固定後に調整
6. **チェストストラップを調整**: ショルダーが肩からずり落ちるのを防ぐ
7. **ロードリフターを調整**: ザックを体に引き寄せて重心を安定させる

### 6.2 腰ベルト（ヒップベルト）の重要性
- **荷重分散**: バックパックの荷重の大部分（約70〜80%）を腰で支える
- **正しい位置**: へその2cmほど下、骨盤の最も出っ張った部分（腸骨稜）を包み込むように締める
- **背面長との関係**: 腰ベルトの位置が著しく上過ぎたり下過ぎたりする場合は、ザックの背面長が合っていない可能性
- **参考情報**: 「腰ベルトは、バックパックの荷重の大部分（一般的に70～80%）を腰で支えるための最も重要な部分です。」<details><summary>出典</summary>`https://yamap.com/magazine/`</details>

### 6.3 各ストラップの役割

| ストラップ | 役割 | 調整のポイント |
|------------|------|----------------|
| ヒップベルト | 荷重の70〜80%を腰で支える | へその2cm下、腸骨稜を包み込むように |
| ショルダーハーネス | 肩とザックを密着させる | 重さが肩にかかりすぎないよう調整 |
| チェストストラップ | ショルダーがずり落ちるのを防ぎ、揺れを抑える | 鎖骨の少し下、締めすぎ注意 |
| ロードリフター | ザックを体に引き寄せ、横揺れを防ぐ | 重心を安定させるために調整 |

- **参考情報**: 「チェストストラップはショルダーハーネスが肩からずり落ちるのを防ぎ、バックパックの揺れを抑える役割があります。」<details><summary>出典</summary>`https://yamap.com/magazine/`</details>

### 6.4 背面長の確認
- **背面長**: 頸椎（首の付け根の出っ張った骨）から腰骨の末端までの長さ
- ザック選びの際に自分の体型に合ったものを選ぶことが重要
- 背面長を調整できるモデルもある
- **参考情報**: 「背面長は、頸椎（首の付け根の出っ張った骨）から腰骨の末端までの長さで測られます。」<details><summary>出典</summary>`https://yamahack.com/`</details>

## 7. 初心者がやりがちな失敗

### 7.1 重心のバランスが悪い
- 重い荷物をザックの下部や背中から離れた位置に詰め込む
- → 重心が不安定になり、歩行時のバランスを崩しやすくなる
- → 肩や腰に過度な負担がかかり疲労の原因に
- **参考情報**: 「重い荷物をザックの下部や背中から離れた位置に詰め込むと、重心が不安定になり、歩行時のバランスを崩しやすくなったり、肩や腰に過度な負担がかかり疲労の原因となります。」<details><summary>出典</summary>`https://yamap.com/`</details>

### 7.2 荷物の取り出しにくさ
- 使用頻度の高いものをザックの奥底に詰め込む
- → 必要な時にすぐに取り出せない
- → 休憩時に他の登山者を待たせてしまったり、行動時間が削られる
- **参考情報**: 「使用頻度の高いものをザックの奥底に詰め込んでしまうと、必要な時にすぐに取り出せず、行動時間が削られたりする原因になります。」<details><summary>出典</summary>`https://yamap.com/`</details>

### 7.3 防水対策の不足
- 突然の雨でザックの中の荷物が濡れる
- → 着替えや寝袋が濡れると体温低下や不快感
- → 時には危険を伴う

### 7.4 詰め込みすぎ・不必要な荷物
- 「あれもこれも必要かも」と必要以上の荷物を持ってくる
- → ザックが重くなり、体力の消耗が早まる
- **ポイント**: 事前に持ち物を選別し、本当に必要なものだけを厳選する

### 7.5 荷物がザックの中で動く
- パッキングが甘く荷物が固定されていない
- → 歩くたびに荷物が揺れ動き、重心がずれる
- → 歩きにくくなる・不快感

## 8. 日帰り登山のパッキング

### 8.1 荷物重量の目安
- 総荷物重量（食料・水分込み）は**体重の1/6以内**、多くても1/5以内が理想
- 例: 体重60kgの場合、10kg程度
- **参考情報**: 「日帰り登山における総荷物重量（食料・水分込み）は、体重の1/6以内、多くても1/5以内が理想とされています。」<details><summary>出典</summary>`https://sotolog-360.com/`</details>

### 8.2 ザック容量
- 日帰り登山には**20L〜30L**程度の容量が適切
- **参考情報**: 「日帰り登山には20L～30L程度の容量が適しています。」<details><summary>出典</summary>`https://mt-inn.jp/`</details>

### 8.3 軽量化のポイント
- 持ち物リストを作成し本当に必要なものか一つずつ検討
- 一つのアイテムで複数の役割を果たすものを選ぶ（兼用品の活用）
- 細かいアイテムの重さも集まるとかなりの重量になる
- 安全に関わる必需品（ファーストエイドキット、地図、コンパス、ヘッドランプなど）は必ず携行

## 9. テント泊のパッキング

### 9.1 ザック容量
- 一般的に**45L〜65L**程度の大型バックパックが推奨
- 初心者には60L〜70Lが安心
- **参考情報**: 「一般的に45L～65L程度の大型バックパックが推奨されます。初心者の方には60L～70Lのものが安心とされています。」<details><summary>出典</summary>`https://montbell.jp/`</details>

### 9.2 主な持ち物リスト

| カテゴリ | 持ち物 |
|----------|--------|
| 行動中の装備 | 登山靴、レインウェア（上下）、防寒着、グローブ、ザックカバー、ファーストエイドキット、行動食、健康保険証コピー |
| 宿泊用の装備 | テント本体、ペグ、グランドシート、シュラフ（寝袋）、スリーピングマット、ヘッドライト、着替え |
| 炊飯用の装備 | バーナー、燃料、クッカー、カトラリー、食料、水筒やウォーターキャリー |
| その他 | 地図、コンパス、スマートフォン、モバイルバッテリー、日焼け止め、トイレットペーパー、ビニール袋、速乾性タオル |

### 9.3 配置のポイント
- **テント**: 背中側・上部に配置（重いため）
- **シュラフ**: ザックの底部に配置（使用頻度が低いため）
- **クッカー・食料**: 背中側・中〜上部に配置
- **レインウェア・行動食**: 雨蓋やサイドポケットに配置（頻繁に使用するため）

## 10. パッキングが登山に与える影響（科学的根拠）

### 10.1 重心と体への負担
- 重心が体の重心に近いほど、体への負担が軽減される
- 重いものが背中から離れた位置にあると、体が後ろに引っ張られる感覚になり、バランスを崩しやすい
- これは物理学的にもモーメント（回転力）の原理で説明できる

### 10.2 荷重分散のメカニズム
- 適切に調整されたヒップベルトにより、荷重の70〜80%を腰（骨盤）で支える
- 肩への負担を大幅に軽減し、疲労を抑える
- ショルダーハーネスは残りの20〜30%を支える役割

### 10.3 平坦な道とクライミングの違い
- **平坦な道**: 荷重を背中の上部から肩あたりに集中させることで、自然な前傾姿勢を補い、前後へのバランスを崩すのを防ぐ
- **クライミング**: 重心を背中の中央あたりに置くことで、体を動かしやすくする

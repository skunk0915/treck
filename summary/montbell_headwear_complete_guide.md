# モンベル 帽子・キャップ完全ガイド サマリ

## 1. 概要
モンベルのヘッドウェアは、登山やキャンプなどのアウトドアシーンから日常使いまで対応する幅広いラインナップが特徴。機能性（防水、透湿、遮光、保温など）とコストパフォーマンスの高さで評価されている。

## 2. 種類と特徴
### ハット (Hat)
- **特徴**: 全周につばがあり、顔だけでなく首筋まで日差しや雨から守る。
- **用途**: 登山、トレッキング、ハイキング。日差しが強い場所や雨天時に最適。
- **代表モデル**:
    - **GORE-TEX ストームハット**: 防水透湿性に優れる。
    - **WIC.ライト ハット**: 通気性と速乾性に優れる。
    - **サウスリム ハット**: バリスパン素材で耐久性が高い。

### キャップ (Cap)
- **特徴**: 前方につばがあり、視界を確保しつつ顔の日差しを防ぐ。
- **用途**: トレッキング、ランニング、日常使い。風で飛ばされにくい。
- **代表モデル**:
    - **ステンレスメッシュ キャップ**: 遮光性と通気性が高く、夏に最適。
    - **GORE-TEX O.D.キャップ**: 防水透湿性があり、雨天対応。
    - **クリマプラス200 O.D.キャップ**: フリース素材で保温性が高い。
    - **リバーシブル バードビルキャップ**: 2つのデザインを楽しめる。

### ビーニー / ニット帽 (Beanie)
- **特徴**: つばがなく、頭部全体を保温する。
- **用途**: 冬山登山、スキー、スノーボード、秋冬のキャンプ、ファッション。
- **素材**: メリノウール、アクリル混紡、シルクなど。
- **代表モデル**: チベタンキャップ（耳当て付き）など。

### その他
- **サンブロック スクリーン**: キャップやハットに取り付けて首筋の日焼けを防ぐ。
- **レインハット/キャップ**: 雨天専用の防水モデル。

## 3. 素材と機能
### GORE-TEX (ゴアテックス)
- **機能**: 世界最高レベルの防水透湿性。
- **メリット**: 雨を防ぎつつ、汗の蒸れを逃がす。全天候対応。

### Wickron (ウイックロン)
- **機能**: 吸水速乾、通気性、UVカット、消臭。
- **メリット**: 汗をかいてもサラッとした肌触り。夏場や運動量の多いシーンに最適。コットンのような風合い。

### ステンレスメッシュ
- **機能**: 遮光性、通気性。
- **メリット**: 太陽光を反射し、熱がこもらない。真夏の炎天下で威力を発揮。

### KAMICO (紙糸プラス)
- **機能**: 通気性、吸放湿性、軽量。
- **メリット**: 清涼感があり、蒸れにくい。日本の高温多湿な夏に合う。

### クリマプラス (フリース)
- **機能**: 保温性、通気性、速乾性。
- **メリット**: 暖かく、濡れても乾きやすい。秋冬の定番。

### バリスパン
- **機能**: 高強度、耐久性。
- **メリット**: 藪漕ぎなどハードな使用に耐える。

## 4. 選び方のポイント
### シーン別
- **登山・トレッキング**:
    - 夏・晴天: 通気性と遮光性重視（ステンレスメッシュ、Wickron）。ハットなら首の日焼けも防げる。
    - 雨天・悪天候: 防水透湿性重視（GORE-TEX）。
    - 冬・寒冷地: 保温性重視（ウールニット、フリース）。
- **キャンプ**:
    - 焚き火: 火の粉に強い素材（コットン、難燃素材）や、リラックスできる締め付けの少ないもの。
    - 夏キャンプ: 虫除けや日差し対策（ハット、サンブロックスクリーン）。
- **ランニング・運動**:
    - 通気性とフィット感重視（メッシュキャップ、WIC.クールキャップ）。
- **日常使い**:
    - デザインと肌触り重視（コットンキャップ、ニット帽、中折れハット）。

### サイズ選び
- S, M, L, XLの4サイズ展開が基本。
- 指1本入るくらいの余裕があるサイズが目安。
- 背面のバックルやドローコードで調整可能なモデルが多い。

# Pa'lante Packs 調査サマリ

## 1. ブランド概要と歴史

### 創業
- **創業者**: Andy Bentz（アンディ・ベンツ）とJohn Zahorian（ジョン・ザホリアン）
- **創業年**: 2016年
- **創業のきっかけ**: 2人はContinental Divide Trail (CDT)で出会った友人。シンプルでウルトラライト、クリーンなパックを作ることを目標に創業
- **初回生産**: 2016年に40個のパックを製造し、Instagram、YouTube、r/Ultralightで販売開始
- **成長**: 6ヶ月以内に需要が高まり、外注製造が必要に。現在はユタ州オグデンで製造

### ブランド名の由来
- "Pa'lante"はスペイン語の"para adelante"（前へ、進み続ける）の短縮形
- Andyがパナマに住んでいた時に学んだ言葉
- 速く軽く、風景の中を進み続けるハイキングスタイルを反映

### 創業者の経歴
- John Zahorianは三冠（PCT、CDT、AT）、Arizona Trail、Colorado Trail、Great Divide Trail、さらにPCTで追加1900マイルをハイク

## 2. デザインフィロソフィーと革新性

### 核となる設計思想
- **目標**: ヒップベルトが不要なほど軽量なパックを作ること（Andy Bentzの言葉）
- **ターゲット**: FKT（Fastest Known Time）スタイル、ファストパッキング
- **特徴**: ショルダーストラップポケット、ボトムポケット、外側メッシュポケット

### Pa'lanteの革新
- ボトムポケット、ソウインのショルダーストラップポケット、外側メッシュポケットなど、現在多くのULバックパックブランドが採用している機能を先駆けて導入し、より広いオーディエンスにアクセス可能にした

## 3. 各モデル詳細スペック

### V2（主力モデル）

**基本スペック**
- **容量**: 31L（16インチトルソ）/ 37L（19インチトルソ）
- **重量**:
  - Gridstop版: 17.7oz（16"）/ 18.6oz（19"）（約500g / 527g）
  - Ultraweave版: 18.1oz（16"）/ 18.7oz（19"）（約513g / 530g）
- **価格**: $240〜$330（構成による）
  - Joey straps付き: $240/$300
  - Pad + sleeve付き: $270/$330
- **素材**:
  - 210D UHMWPE Gridstop または Ultraweave 200x
  - 210D 2x2 UHMWPE grid mesh
- **適正荷重**: 約25ポンド（11kg）以下で最も快適
- **トルソ長**: 19インチ（48cm）

**特徴**
- ボトムポケット（UHMWPE grid mesh製）- 歩きながらスナックにアクセス可能
- ショルダーストラップポケット（高さ6.5インチ）- スマホ、カメラ、スナック用
- 大型サイドポケット - 1Lボトル2本まで収納可能、歩きながらアクセス可能
- 収納可能なヒップベルト
- ロールトップクロージャー
- 調整可能なサイド圧縮ストラップ
- テントステークポケット（フロントメッシュポケットに統合）
- サイドテンションロック ショックコード - 高いギアの固定、圧縮、乾燥用
- トップ圧縮ストラップ（アルミG-hook使用）

**評価**
- Backpacking Lightより「Highly Recommended」評価
- フレームレスパックなので、パッキングにコツが必要
- ベースウェイト12ポンド以下の経験豊富なULハイカー向け
- パッキングをしっかり行えば体にフィットし、体の一部のように感じられる

### Desert Pack（大容量モデル）

**基本スペック**
- **容量**: 43L
- **重量**:
  - Gridstop版: 18.8oz / 19.2oz（約533g / 544g）
  - Ultraweave版: 20.9oz / 21.6oz（約592g / 612g）
- **素材**:
  - 210D UHMWPE Gridstop または Ultraweave 400x
  - 210D 2x2 UHMWPE grid mesh
- **トルソサイズ**: 16-17インチと19インチの2サイズ展開

**用途**
- より長く、重い荷物を運ぶために設計
- 水をたくさん運ぶ必要がある場所（砂漠など）
- V2より容量と耐久性を重視
- 日本では水場が少ない縦走路や長期縦走に最適

**特徴**
- ULバックパックへの「優れたエントリーポイント」として評価
- V2よりやや重いが、より多くの荷物を運べる

### Joey（ファストパック/ベスト型）

**基本スペック**
- **容量**: 24L（内部容量）
- **重量**:
  - Gridstop版: 14.3oz（約405g）
  - Ultraweave版: 14.4oz（約408g）
- **素材**:
  - 210D UHMWPE Gridstop または Ultraweave 200x
  - 70D ナイロンリップストップ
  - 210D 2x2 UHMWPE grid mesh
- **サイズ**: 18インチトルソ長のワンサイズ

**用途**
- FKT（Fastest Known Time）、オーバーナイトハイク、ファストパッキング向け
- マルチデイ用のベストスタイルパック
- トレイルランニングにも対応

**特徴**
- ベスト型ハーネスで揺れを極限まで抑制
- ショルダーポケット装備
- ウルトラメッシュ生地
- 伸縮性のあるボトムポケット
- 日帰り〜極限まで切り詰めたオーバーナイトハイクまで対応

**評価**
- 非常にコンパクトで軽量
- 走れるほどの安定性

### Simple Pack / Simple Pack Kit

**概要**
- V2の前身モデル
- 現在はV2が主流だが、一部でDIYキットとして販売
- V2と同様の基本設計だが、機能が簡素化されている

## 4. 素材と製造

### 使用素材
- **UHMWPE (Dyneema) Gridstop 210D**: 軽量で強度が高い主要素材
- **Ultraweave 200x / 400x**: Challenge Sailcloth製の高性能素材。Gridstopより高価だが耐久性と防水性が向上
- **UHMWPE grid mesh 210D 2x2**: メッシュポケット用
- **70D ナイロンリップストップ**: Joeyの一部に使用
- **PUコーティングナイロンリップストップ**: 防水性向上のため

### 製造
- 現在はユタ州オグデンで製造
- 高品質な補強ステッチ、バータック、ハードウェアで強度と長寿命を確保

## 5. 耐久性とメンテナンス

### 耐久性
- ミニマリストのフレームレスパックとしては「驚くほど耐久性が高い」と評価
- 210D UHMWPE Gridstop本体はスルーハイキングの過酷さに耐える
- Ultraweave 200X版はより高価だが、さらに耐久性と防水性が保証される
- 補強ステッチ、バータック、高品質ハードウェアで全体的な強度を向上

### 注意点
- Ultraweave素材の外側ポケットは早期に摩耗の兆候を示す場合がある
- 清掃が難しい

### 防水性
- Gridstop版: わずかに防水性があるが完全防水ではない。パックライナー必須
- Ultraweave版: より防水性が高いが、激しい豪雨では浸水する可能性あり
- いずれもメインコンパートメントに防水ゴミ袋またはパックライナーを使用することを推奨

### メンテナンス
- 具体的なメンテナンス方法や清掃手順についての詳細情報は見つからず
- Ultraweave素材は清掃が難しいとの指摘あり

## 6. 他ブランドとの比較

### vs Hyperlite Mountain Gear

**防水性と素材**
- **Pa'lante V2**: 最小限の防水性、パックライナー必須。快適な生地と最大限の軽量化を優先
- **Hyperlite**: Dyneema Composite Hybrid (DCH)生地による優れた防水性。コーティングや処理に頼らず、経年劣化しにくい
- **Hyperlite Unbound 40**: トップクラスの構造品質

### vs Gossamer Gear

**共通点**
- **Pa'lante V2とGossamer Gear Mariposa 60/Gorilla 50**: いずれも最小限の防水性でパックライナー必須
- どちらも快適な素材と軽量な構造を重視

**相違点**
- **Gossamer Gear Fast Kumo vs Pa'lante V2**:
  - Kumoもラージキャパシティパックで、どちらもパッド入りショルダーストラップあり
  - Kumoのストラップはランニングベスト型で、2つのスターナムストラップ装備
  - Kumoは価格がV2より大幅に安い

### 歴史的な位置づけ
- **Gossamer Gear**: ULパックにフレームを追加
- **Hyperlite Mountain Gear**: アイスクライミングやパックラフティングなど幅広いアウトドア活動向けの軽量ギアを強調
- **Pa'lante**: 特定のハイキングスタイル（長く、速く）に焦点を当てた

## 7. スルーハイカーの実際の使用経験

### PCT（パシフィック・クレスト・トレイル）での使用

**2016年以降の使用実績**
- Pa'lanteパックは2016年からスルーハイクで使用され、思慮深いデザインとV字型構造が評価されている

**ハイカーのフィードバック**
- 一部のハイカーは、PCTには37L V2ではなく43L Desert Packを選べばよかったと後悔
  - 4ヶ月のハイクでより多くの柔軟性が得られたとのこと
- あるハイカーはPCTの最終週にPa'lante V2 Gridstopに切り替え
  - 理由: ハイキング中に食べやすいボトムスナックポケット
- V2は総重量約25ポンドまで快適、Desert Packはさらに多くの重量に対応可能

### CDTとの関係
- 共同創業者John Zahorianは三冠達成者（PCT、CDT、AT）
- さらにArizona Trail、Colorado Trail、Great Divide Trail、PCTで追加1900マイルをハイク
- Andrew BentzとJohn ZahorianはCDTで知り合った友人

### 一般的な評価
- ベースウェイト12ポンド以下の経験豊富なULスルーハイカー向けのトップクラスのフレームレスパック
- PCT Daysで人気のある小型ULパックの一つ（LiteAFと並んで）

## 9. まとめ：Pa'lanteの特徴

### 強み
1. **革新的なデザイン**: ボトムポケット、ショルダーストラップポケットなど、現在業界標準となった機能を先駆けて導入
2. **超軽量**: 全モデルが500g前後という驚異的な軽さ
3. **機能的**: 歩きながらアクセス可能なポケット配置で、立ち止まる時間を削減
4. **耐久性**: 軽量ながら補強ステッチと高品質素材で耐久性を確保
5. **選択肢**: V2（標準）、Desert Pack（大容量）、Joey（ファストパック）と用途に応じた選択が可能

### 注意点
1. **フレームレス**: パッキングにコツが必要、経験者向け
2. **適正荷重**: 基本的に11kg以下の軽量装備が前提
3. **入手困難**: 常に品薄で、見つけたら即購入が必要
4. **完全防水ではない**: パックライナー必須
5. **価格**: $240〜と決して安くはない

### 適したユーザー
- ベースウェイト12ポンド（約5.4kg）以下のULハイカー
- パッキングスキルのある経験者
- 長距離・高速ハイキングを目指す人
- FKTやファストパッキングに挑戦する人

# 初めての山選びガイド【初心者向け山の選び方】サマリ

## 1. 初心者の山選び基本基準

### 1.1 コースタイムの目安
- **目安は4時間前後まで**: 休憩時間含まず合計4時間前後。休憩込みで約6時間程度の山行が無理なく日帰り可能。
- **日帰り5時間以内**も推奨される
- **注意**: ガイドブックのコースタイムには休憩時間が含まれていない。1〜2割増しで見積もるべき。
- **早めの出発、早めの下山**を習慣に

**参考情報**: 「登山初心者の場合、休憩時間を含めずに合計4時間前後のコースタイムの山を選ぶのがおすすめです」<details><summary>出典</summary>`https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQHV5oxH6moQwfYHzeM_ySq2d-iSYRScCM4UYJwhPP_rZyuce8qsbU3lqMSdakJYpmvBND_ynVkiQQGe6dQKBVczzkKa654ke4TzkoGQ9MsrgDiF-TOveIO0ZLP3ucI6s_lgOCqjONGrhtjrA51jf7U0Cgnu`</details>

### 1.2 標高差の目安
- **目安は500〜600m**: 登山口から山頂までの標高差が600m前後が無理なく楽しめる
- **1時間あたりの登りの目安**: 初心者が無理なく登れるペースは1時間あたり300m程度、下りは400m程度
- **低山でも標高差に注意**: 山の標高だけでなく、登山口からの標高差を確認することが重要
- **標高1,000m以下**が一般的に低山と呼ばれ初心者向け

**参考情報**: 「登山口から山頂までの標高差が600m前後の山が、初心者にとって無理なく楽しめる目安」<details><summary>出典</summary>`https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQGN_RYOLs6gpWRVWGe4ydJmJ9HKwe_lS-BI8HyScOiefe4RpeAfValIvo72rpAsocz7Ye0sRttKUfTkz8GGGMVXiigMAvF6ljSSE8t2gzekC7m0EntpaiHyN7Zwsgk_1iwRRw==`</details>

### 1.3 その他の重要な選定基準
- **整備された登山道**: 道迷いのリスク軽減、標識が明確
- **登山者が多い山**: メジャーな山は道が明確で万が一の際も助けを得やすい
- **アクセスの良さ**: 公共交通機関でアクセス可能、ケーブルカー・リフト利用可能
- **往復型（ピストン）コース**: 体調不良や天候悪化時に引き返しやすい
- **危険箇所が少ない**: 岩場、鎖場、細い尾根などが少ないコース
- **森林限界を超えない山**: 天候変化が激しくなりにくい

**参考情報**: 「登り下りで同じ道を歩く『往復型（ピストン）』は、天候悪化や体調不良の際に引き返しやすく、初心者におすすめのコース形式」<details><summary>出典</summary>`https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQEojOcNT09ab9acWLpedtR-gnHrP-9xcAxprCTqsRbTHjkU8vV5SBdk8T0ZSbwdXAqREhinaqOMYT_88pHAcybKQPWE5kntf8IEq3klvH2Bfx5Nbax5qvYv-5wi`</details>

## 2. 山のグレーディング

### 2.1 グレーディングとは
- 2014年に長野県が初めて「信州 山のグレーディング」を公表
- 目的: 登山者が自分の力量に合った山を選べるようにし、遭難事故を防止
- 現在は長野県、山梨県、静岡県、新潟県、岐阜県、栃木県、群馬県、山形県、秋田県など多くの県で公開

**参考情報**: 「長野県が2014年に初めて『信州 山のグレーディング』を公表しました。これは、登山ブームにより経験不足の登山者や体力を過信した中高年による遭難事故が増加したことを受け導入されました」<details><summary>出典</summary>`https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQE03vEvDfFGkuCwb1oC7tgHQ1SJfhC6DVKrA440FTjWtdSZo8UgZEIAIlpfb5x4uxmClx6EVe8xO6jmyayCGKYm2f7yzhARG4ChRkvUVDQiFT0aaNpWIBrfRzPVPQzYSnmGD_9qwcXYb98CYyYvWi3wnWBRY98icFOoaMN4c7AIAeFRaN0VH3AgKj_M3tmVP9BA1yIn3vvrlN6mfJz2tBnBQvAZ4-7VGYPcMA==`</details>

### 2.2 体力度（10段階）
- 数字が大きいほど必要な体力が高い
- 算出要素: 標準コースタイム、ルート全長、累積登り標高差、累積下り標高差
- **目安**:
  - 体力度1〜3: 日帰り可能
  - 体力度4〜5: 1泊以上が望ましい
  - 体力度6〜7: 2泊以上必要
  - 体力度8〜10: 2〜3泊以上必要

### 2.3 技術的難易度（A〜E）
| レベル | 説明 |
|--------|------|
| A（初級） | 整備された登山道、転落・滑落の危険性低い、道迷いの心配少ない |
| B（初級〜中級） | 沢渡りや急な坂あり、道が分かりにくい箇所がある |
| C（中級） | はしごや鎖場あり、転落・滑落の危険がある箇所が出現 |
| D（上級） | 厳しい岩場、不安定なガレ場、はしご、鎖場、痩せ尾根などが出現 |
| E（最難関） | ロープワークなど専門的な登山技術が必要 |

### 2.4 初心者向けグレード
- 体力度1〜3、技術的難易度A〜Bに該当する山が初心者向け
- 高尾山（表参道ルート）: 体力度2、技術的難易度A

**参考情報**: 「体力度1～3、技術的難易度A～Bに該当する山が初心者向けとされています」<details><summary>出典</summary>`https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQF6yOCgA32re_Ytm2JN5m3nZdLR3tcZmnnpFCQJR00J2vdRspnGzPAsxc2Z2DyvgtbBppvTypXqi0HkHYDIBLHehypIxBV1NW5lWR6-9MTYCa1gcOlKmSi8VT9xYfkUaX14DBkOhnL3`</details>

### 2.5 注意点
- グレーディングは**積雪のない時期で天候が良好な条件**での評価
- 悪天候、残雪、体調、その他偶発的要因によるリスクも考慮が必要

## 4. 山小屋vsテント泊

### 4.1 初心者には山小屋泊がおすすめ

**山小屋泊のメリット**:
- **荷物の軽量化**: テント、寝袋、マット、調理器具、食料を持つ必要なし
- **安全な環境と休息**: 悪天候時も風雨をしのげる
- **食事の提供**: 夕食・朝食が提供される
- **設備**: トイレ整備、暖房・電気のある山小屋も
- **緊急時の安心感**: トラブル時に対応可能

**山小屋泊のデメリット**:
- 宿泊費用が高い
- 消灯時間、食事時間などルールあり
- 相部屋が多くプライベート空間少ない
- 人気山小屋や連休中は混雑、事前予約必須
- 他の宿泊者のいびきなどで眠りを妨げられる可能性

## 7. 季節ごとの登山

### 7.1 春（3月〜5月）
**メリット**: 桜、新緑、花が楽しめる。虫が少ない。日が長くなる。
**注意点**: 天候急変しやすい、山頂は気温低い、雪解けによるぬかるみ、動物との遭遇注意

### 7.2 秋（9月〜11月）
**メリット**: 紅葉が美しい、虫が少ない、空気が澄む
**注意点**: 日没が早い（つるべ落とし）、寒暖差が大きい、高山では雪も、落ち葉で滑りやすい

**参考情報**: 「秋は日没の時間が急激に早まるため、『つるべ落とし』と呼ばれる秋の日は行動時間に注意が必要」<details><summary>出典</summary>`https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQFzZcy40v4EoXigg6iSkbTgK63gj0sKhtmVdcEexRauFPIi8XvCyKf3AJwsUbCovxel7KQnm892TSTU4NCxH1IHjwPqNRzhWFNqznINtiVg94CM`</details>

### 7.3 梅雨時期
**リスク**: 低体温症、視界不良、体力消耗、ぬかるみ・スリップ、川の増水
**対策**: 雨具をすぐ出せる場所に、こまめな水分・エネルギー補給

### 7.4 夏（6月後半〜9月）
**メリット**: 富士山など高山の開山期
**注意点**: 熱中症・脱水、天候急変（雷雨・集中豪雨）、高山での低体温症、紫外線、毒虫・毒草

## 8. 初心者がやりがちな失敗と注意点

### 8.1 準備・計画不足
- **レベルに合わない山選び**: 人気や景色だけで選ばない
- **不適切な服装**: 綿素材は汗で乾きにくく体温低下の原因。吸汗速乾性のあるウェアを

### 8.3 体調・身体トラブル
- **疲労・筋肉痛**: 下山時の急な下りでの大股歩きが原因
- **靴擦れ・マメ**: 事前にテーピングで対策
- **高山病**: 水分補給をしっかり、症状が出たら無理せず下山
- **熱中症・低体温症**: 気温変化に対応できないと発生

- **単独行動**: 初心者は経験者と、またはツアー参加が安全

## 9. まとめ：初心者の山選びチェックリスト

1. **コースタイム**: 4時間前後以内
2. **標高差**: 500〜600m以内
3. **登山道の状態**: 整備されている、登山者が多い
4. **アクセス**: 公共交通機関で行ける、ケーブルカー・リフトあれば尚良し
5. **コース形式**: 往復型（ピストン）が安心
6. **グレーディング**: 体力度1〜3、技術的難易度A〜B
7. **天気予報**: てんくら等で登山指数確認
8. **地図アプリ**: YAMAP等で地図を事前ダウンロード
9. **装備**: 適切な服装、レインウェア、ヘッドライト等
10. **登山計画書**: 作成し提出・共有

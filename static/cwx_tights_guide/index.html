第1章: CW-Xとは何か?「履くテーピング」の誕生秘話

1.1 スキーでの怪我から生まれたアイデア

CW-Xは、ワコールの人間科学研究所が開発したブランドだ。1991年、ワコールの研究員がスキーで怪我をした際、テーピングで関節を固定したことがきっかけだった。「このテーピングの効果をタイツに組み込めないか?」というアイデアから開発が始まったんだ。
出
「1991年にワコールの研究員がスキーでの怪我から着想を得た「テーピング原理をタイツに組み込む」というアイデアに基づいて開発されました。」(https://motocampers.net)
へえ!怪我がきっかけだったんですね。でも、タイツにテーピングってどういうことですか?
タイツの生地に、伸縮性のある特殊なラインを縫い込んであるんだ。このラインが、筋肉や関節を適切な方向にサポートする。まるで、プロのトレーナーが常にテーピングしてくれているような感覚だ。

1.2 ワコール人間科学研究所の科学的アプローチ

京都にあるワコール人間科学研究開発センターでは、人間の動きを徹底的に研究している。ランニング中の膝の動き、着地時の衝撃、筋肉の揺れ方など、すべてデータ化して分析しているんだ。
出
「京都にあるワコール人間科学研究開発センターでは、人間の動きの科学(運動学)を詳細に研究しており、その成果がCW-Xのコンプレッションギアに活かされています。」(https://cw-x.com)
なんか本格的ですね…。ただのスポーツウェアじゃないんだ。
そうだ。生体力学的研究により、以下の効果が科学的に証明されている。
  1. ランニング中の体幹・下肢の安定性とバランスの向上
  2. 足が地面に着地する際の膝への衝撃軽減
  3. 運動中および運動後の疲労発生率の低下
出
「生体力学的研究により、ランニング中の体幹・下肢の安定性とバランスの向上、足が地面に着地する際の膝への衝撃軽減、運動中および運動後の疲労発生率の低下が示されています。」(https://cw-x.com)
科学的根拠があるんですね。でも、高そう…。
確かに高い。1万5千円以上する。だが、膝を痛めて病院に通うコストや、痛くて山を楽しめない損失を考えれば、安い投資だと私は思う。

第2章: 3つのモデルの違い - あなたに合うのはどれ?

CW-Xって、種類がいっぱいあるんですよね?どれを選べばいいんですか?
主に3つのモデルがある。「エキスパート」「スタビライクス」「ジェネレーター」だ。サポートする範囲と強度が違う。順番に説明しよう。

エキスパートモデルは、CW-Xの基本モデルだ。股関節、太もも、膝、ふくらはぎをサポートする。特に、ランナーが痛みを感じやすい膝のサポートに重点を置いている。

これが一番安いんですか?
そうだ。初めてサポートタイツを試す人や、軽い登山・ハイキングなら、これで十分だ。ただし、腰のサポートはないので、長時間の登山には物足りないかもしれない。

2.2 スタビライクスモデル - 「腰+膝」のミディアムサポート

スタビライクスモデルは、エキスパートに「腰のサポート」を追加したモデルだ。腰、股関節、膝をサポートする。

腰のサポートって、そんなに重要なんですか?
非常に重要だ。長時間歩くと、腰が疲れて体の軸がブレる。すると、膝や足首に余計な負担がかかる。腰を安定させることで、全体のバランスが良くなり、疲れにくくなるんだ。
なるほど!じゃあ、登山ならこれがいいんですね。

ジェネレーターモデルは、CW-Xの最上位モデルだ。腰、股関節、お尻、膝、太もも、ふくらはぎ…下半身全体をフルガードする。

第3章: テーピング原理の科学 - なぜ膝が守られるのか?

で、結局どういう仕組みで膝が守られるんですか?
良い質問だ。CW-Xの核心部分を説明しよう。

3.1 膝蓋骨(膝のお皿)のブレを抑える

下り坂で足をついた瞬間、膝のお皿(膝蓋骨)が上下左右にブレる。これが痛みの原因の一つだ。CW-Xのサポートラインは、お皿を下から持ち上げるように固定し、ブレを防ぐ。
出
「着地時の衝撃軽減: CW-X独自のテーピング原理により、着地時の衝撃から膝を保護します。」(https://runnet.jp)
お皿が動くんですか?知らなかった…。
実際に履いて下ると、「膝がガクッと崩れない」感覚がある。着地の衝撃を筋肉ではなく、タイツの張力が受け止めてくれるイメージだ。

3.2 筋肉の無駄な揺れを抑制

歩くたびに太ももの肉がブルンブルンと揺れる。これが意外とエネルギーを消耗するんだ。着圧で筋肉をギュッと押さえ込むことで、無駄な揺れをなくす。
出
「着圧によって筋肉の無駄な動き(揺れ)が抑えられることで、エネルギーロスが減少し、疲労の蓄積が抑制されます。」(https://eightyuruspolifeblog.com)
筋肉の揺れって、そんなに疲れるんですか?
そうだ。長距離を歩いた後の「足のだるさ」が劇的に減る。テント場に着いた時、足がパンパンになっていないのは感動的だぞ。

3.3 血流促進とリカバリー効果

ふくらはぎに適度な圧力をかけることで、下半身に溜まった血液を心臓に戻すポンプ作用を助ける。これにより、足のむくみが軽減される。
出
「コンプレッションは血流と酸素供給を促進し、筋肉からの乳酸の除去を効率的に行います。」(https://eightyuruspolifeblog.com)
むくみが取れるのは嬉しい!美脚効果もありそう。
そうだな。ただし、あくまでスポーツ用だ。日常使いには向かない。

CW-Xのタイツは、生地がしっかりしているため、夏場や気温が高い環境では暑く感じる。汗をかきやすく、蒸れが気になるという声もある。

それは困りますね…。トイレ我慢しちゃいそう。
我慢はするな!早めに行け!山行などで使用する際には、テントや小屋に着いたらすぐに脱いで通常のパンツに履き替えるという利用者もいる。

5.3 履くのが大変!ストッキングのように慎重に

CW-Xは、そのサポート機能を最大限に発揮させるために、正しい位置に着用することが非常に重要だ。ストッキングを履くようにゆっくりとたくし上げ、膝の皿や関節、筋肉のサポートラインが正確に合うように調整する必要がある。
出
「CW-Xは、そのサポート機能を最大限に発揮させるために、正しい位置に着用することが非常に重要です。このため、着用時にはストッキングを履くようにゆっくりとたくし上げ、膝の皿や関節、筋肉のサポートラインが正確に合うように調整する必要があります。」(https://cw-x.jp)
めんどくさそう…。
無理に引っ張ると生地を傷める可能性もあるため、爪を立てずに慎重に着用することが推奨されている。朝、時間に余裕を持って準備することが大切だ。

第6章: 長く使うための洗濯・メンテナンス術

高いなら、長く使いたいです!洗濯方法とか教えてください。
良い心がけだ。正しいメンテナンスで寿命を延ばせる。

6.1 洗濯の基本ルール

以下のルールを守れ。
  1. 洗濯機を使用する場合は、必ず洗濯ネットに入れる
  2. 洗剤はよく溶かしてから洗濯を始める
  3. バスタオルなどの大きなものと一緒に洗うのではなく、下着などの小さなものと一緒に洗う
  4. 手早く洗い、すすぎは十分にする
  5. 脱水も洗濯ネットに入れたまま軽く行う
  6. 40℃を上限とした弱い洗濯が可能。塩素系漂白剤やタンブラー乾燥(乾燥機)は避ける。陰干しが推奨
出
「洗濯機を使用する場合は、必ず洗濯ネットに入れてください。」「洗剤はよく溶かしてから洗濯を始めてください。」(https://cw-x.jp)
結構デリケートなんですね。
そうだ。ポリウレタン素材を使用しているため、雑に扱うと劣化が早まる。

6.2 修理サービスを活用しよう

CW-X製品に穴が開いたり破れたりした場合、ワコールによる修理サービスを利用できる。修理自体は無料だが、製品を送る際と返送される際の送料は利用者の負担だ。
出
「ワコールはCW-Xスポーツタイツの修理を受け付けています。」「修理自体は無料ですが、製品を送る際と返送される際の送料は利用者の負担となります。」(https://opal.ne.jp)
無料で修理してくれるんですか!良心的ですね。
ただし、機能ラインにかかる破れや、生地が残っていない大きな穴、脆化(生地が薄くなったり伸びが悪くなったりしている状態)などは修理対象外となる場合がある。修理に出す前には、必ず製品を洗濯しておく必要がある。修理には約1ヶ月半程度の期間がかかる。
出
「機能ラインにかかる破れや、生地が残っていない大きな穴、脆化(生地が薄くなったり伸びが悪くなったりしている状態)などは修理対象外となる場合があります。」(https://opal.ne.jp)

CW-Xスポーツタイツの着圧は、使用回数が増えるほど低下していく。週2回の使用を2年間続けると、着圧が大幅に低下する可能性がある。使用頻度にもよるが、サポートタイツの寿命は1〜3年程度とされている。

導入

山でおいしいご飯が食べたい。でも、荷物は軽くしたいし、調理の手間もかけたくない——そんな登山者の願いを叶えてくれるのが「フリーズドライ食品」だ。

特に「尾西食品のアルファ米」と「アマノフーズのフリーズドライ」は、登山者の間で絶大な人気を誇る二大巨頭。どちらも「お湯を注ぐだけ」で本格的な食事が楽しめるが、実は製法も特徴もまったく違う。

この記事では、両者の違いを徹底解説しながら、山での最強の組み合わせを探っていく。

登場人物紹介

登山初心者。おいしいものが大好きで、山でも絶対に妥協したくない派。「軽くておいしい」が最優先。
登山歴20年のベテラン。テント泊縦走を数え切れないほど経験し、山での食事の大切さを身をもって知っている。

なぜフリーズドライ食品が登山で重宝されるのか？

先生、山ご飯って正直めんどくさくないですか？重いし、調理に時間かかるし…。
それを解決するのがフリーズドライ食品だ。水分を抜いて軽量化しているから、持ち運びが楽。しかもお湯を注ぐだけで調理できる。
えー、でもインスタントラーメンとかと何が違うんですか？
大きく違うのは「味・香り・栄養の保持力」だな。

尾西食品のアルファ米 —— 主食の王道

まずは「主食」の代表格、尾西食品のアルファ米から説明しよう。
アルファ米って名前からして謎なんですけど…なんでアルファなんですか？
いい質問だ。生のお米のデンプンは「ベータ化」という消化しにくい状態なんだが、炊飯すると「アルファ化」という消化しやすい状態に変わる。
参
「アルファ米は、生米のデンプンが消化しにくい「ベータ化」の状態であるのに対し、炊飯によって消化しやすい「アルファ化」の状態になったデンプンを、このアルファ化状態を保ったまま乾燥させる技術を利用しています」
出典https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQEI3YWAv-A2dSR1bUr-ZqlCfsyVtItvb_ENj9wgPpTto3UvKpuw7edElSmRiddQJ3kIIHRfCI-MS53XIK-3bbOD_wtLZ6ImcWLxWq9-aVbmJAmQHRvpiqXUrPPuO4k=
ふむふむ…で、それを乾燥させたのがアルファ米？
その通り。炊きたての状態を保ったまま急速乾燥させているから、お湯を注ぐと炊きたてに近いご飯が復活するというわけだ。

参
「お湯または水を注ぐだけで、炊きたてのようなふっくらとしたご飯に戻ります。お湯であれば約15分、水であれば約60分で完成します。」

携帯おにぎりの作り方

さっきのおにぎり、具体的にどう作るんですか？
手順を説明しよう：
  1. 袋を開封し、中の脱酸素剤を取り出す
  2. 袋の注水線まで、お湯または水を約67ml注ぐ
  3. 袋のチャックをしっかり閉め、全体にお湯が行き渡るように20回ほどよく振る
  4. お湯の場合は15分、水の場合は60分待つ
  5. 待機時間が経過したら、袋の切り口に沿って袋を切り開き、そのまま食べる
参
「袋のチャックをしっかり閉め、全体にお湯（または水）が行き渡るように20回ほどよく振ります。」
出典https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQE-8vEboWyCbIFMOfO6xmMoJH7TKCc5rhOB3oBULfogBqZ2BUrYjaE5-tFGYZk-jhGB9nTwDBTZNtKx3E-j86VfF67LkoNTtcc3uVcEkpJvzz8jk_MvqFPyJA6H3EYzXw==
振るんですね！普通のアルファ米と違う。
味は鮭、わかめ、五目おこわ、昆布の4種類。しかも携帯おにぎりは製造から5年半の長期保存が可能だ。

アルファ米とフリーズドライ米の技術的違い

アルファ米の製造方法

参
「炊飯/蒸米工程: 米を炊き上げる、または連続蒸米機で蒸気炊飯することでアルファ化させます。乾燥工程: 炊き上がったご飯を熱風で急速に乾燥させ、アルファ化デンプンを保ちつつ水分を除去します」
出典https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQEI3YWAv-A2dSR1bUr-ZqlCfsyVtItvb_ENj9wgPpTto3UvKpuw7edElSmRiddQJ3kIIHRfCI-MS53XIK-3bbOD_wtLZ6ImcWLxWq9-aVbmJAmQHRvpiqXUrPPuO4k=
アルファ米は以下の工程で作られる：
  1. 原材料の受け入れ: 良質な国産米を使用
  2. 浸漬: 米に十分な水を吸水させる
  3. 炊飯/蒸米工程: 米を炊き上げてアルファ化させる
  4. 乾燥工程: 炊き上がったご飯を熱風で急速に乾燥
  5. テンパリング: 乾燥後、米の水分を均一化
  6. 膨化: 復元時の吸水性を良くする
熱風で乾燥させるんですね。

登山でフリーズドライ食品を使うメリット・デメリット

結局、山でフリーズドライを使うメリットってなんですか？
大きく4つある。

メリット

1. 軽量・コンパクト

これが最大のメリットだ。水分が除去されているから、とにかく軽い。
テント泊で食事3日分とか持っていくと、これだけで違いますよね。
まさにそうだ。特に縦走ではグラム単位で荷物を削るから、フリーズドライは必須だ。

2. 調理が簡単

お湯を注ぐだけ。疲れて帰ってきた夜に、複雑な調理をする必要がない。
めんどくさがりの私にぴったりですね。
調理時間が短いから、燃料も節約できる。

3. 長期保存が可能

食品中の水分量が極めて低いため、酵素や微生物の作用が抑制されて日持ちが良い。
防災用にストックしておいて、賞味期限が近づいたら山で消費するのがいいですね。

4. 味・香り・栄養価の保持

参
「フリーズドライ製法は、食材を凍結させて真空乾燥させるため、元の食品の味、香り、栄養素が残りやすいとされています」
出典https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQEnYG5mkmut56QT0VAMrZwsnB-RQV568fCWztJYffcIUWP0t_-DS6N61DZQwwfViWrKnQrm7i0SLVojTxuzBcyf4vqiRQHbxzI33PZZBrhsG1bYN-USmtYa3ubf51lz981ydGU=
乾燥食品とは思えないほど、味が良い。

デメリット

いいことばっかりじゃないですよね？デメリットも教えてください。
もちろんある。正直に言おう。

1. 水の確保が必要

フリーズドライ食品の調理にはお湯や水が必要だ。登山中は水を運ぶか、現地で確保・沸騰させる必要がある。
水場がない山だと厳しいですね。

2. 塩分量が多い製品がある

参
「一部のフリーズドライ食品は塩分量が多めに含まれている場合があり、行動中に多く摂取する場合は意識的な水分補給が重要です」
出典https://vertexaisearch.cloud.google.com/grounding-api-redirect/AUZIYQEEauIC65XipYhkRqWfefF2_1ByV83fmb2TwoFVtq5uYLMWpufFh2WcebzjAwfmmxypZY3XNAtOAgo4R0sz05rwoFz3dsUX5_GrS0q82RAEXRzyiqzPdMCdjsbr1AwZBhhp7nplfDXNXQhJD-KKXp9C-XftAlLpqWEZmWDrNJsokfE=
行動中に多く摂取する場合は、意識的に水分補給をしないといけない。
あ、だから減塩タイプがあるんですね。

3. カロリー不足の可能性

フリーズドライ食品だけでは、登山に必要な十分なカロリーを摂取できない場合がある。行動食と組み合わせてエネルギーを補給することが大切だ。
ナッツとかエナジーバーとかも持っていかないとダメってことですね。

4. 比較的高価

特にフリーズドライ米はアルファ米の約2倍の価格になることもある。日常使いには少しコストがかかる。

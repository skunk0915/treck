導入

キャンプ用テーブルを探していると、必ず目にするのがDOD（ディーオーディー）のユニークな製品たち。「グッドラックテーブル」「ソトメシンガー」「テキーラテーブル」「キャナリーテーブル」…名前だけでも個性的で目を引きますが、正直「どれがどう違うの？」と混乱している方も多いのではないでしょうか。

この記事では、DODのテーブル製品を徹底解説。収納力重視のグッドラックテーブルから、焚き火で使えるテキーラテーブル、ソロキャンプ向けのソトメシンガーまで、それぞれの特徴と選び方のポイントをわかりやすくお伝えします。ベテラン登山ガイドの先生と初心者JKの会話形式で、あなたにぴったりの一台が見つかるはずです。

登場人物紹介

元気で好奇心旺盛な登山初心者。見た目重視で「形から入る」タイプ。専門用語は苦手で、素朴な疑問や「高い」「重い」といった不満は正直に口にする。
登山歴20年のベテランガイド。無骨で質実剛健な性格だが、道具への愛と知識は深い。初心者にも分かりやすく解説してくれる頼れる存在。

DODテーブルはなぜ種類が多い？

先生、DODのテーブル調べてたんですけど、名前が多すぎて頭が痛くなってきました。グッドラックとかソトメシンガーとか、もはや呪文ですよね？
ははは、確かにDODは製品名にこだわりがあるな。でも、それぞれ明確な用途があるんだ。大きく分けると5つのカテゴリーになる。
5つも！？
まあ落ち着いて聞いてくれ。
  1. 収納重視系: グッドラックテーブル、マルチキッチンテーブル
  2. 焚き火対応系: テキーラテーブルシリーズ
  3. ソロ・調理特化系: ソトメシンガーZ/RX、ダイキチテーブル
  4. 天然木デザイン系: キャナリーテーブル、ジミニーテーブル
  5. 特殊用途系: ワンポールテントテーブル、タナカテーブル
あ、こうやって分類するとちょっと分かりやすいかも。
そうだろう。自分のキャンプスタイルに合わせて選べば、後悔しない買い物ができるはずだ。

収納力抜群！「グッドラックテーブル」

Image Prompt: キャンプサイトでグッドラックテーブルを使っている様子。テーブル下の収納スペースにキャンプギアが整理されている。

まず紹介したいのがグッドラックテーブルだ。ファミリーキャンプで「とにかく散らかる！」という悩みを一発で解決してくれる。
散らかるの分かります！ランタンとか調味料とか、どこに置いたか分からなくなるんですよね。
このテーブルは、天板の下に大容量の収納スペースがあるんだ。しかも両サイドにポケットも付いている。
参
「天板が広く、収納が沢山入ってとにかく便利！」
出典https://review.rakuten.co.jp/item/1/212552_10015614/
へえ、テーブルの下が棚になってるってことですか？
その通り。さらに高さが4段階に調節できる。44cm、55.5cm、69cm、80.5cmと変えられるから、ロースタイルからキッチンテーブルまで対応可能だ。
80.5cmって結構高いですね。立って料理するのに良さそう。
そうだな。そしてこのテーブル、グッドラック機能という面白い使い方ができる。
グッドラック…幸運？
車の荷室に積載する時、折りたたんだテーブルを棚として活用できるんだ。これで荷室に3段の収納スペースを作れる。荷崩れ防止にもなるし、効率的な積載ができる。
え、それすごくないですか!? 帰りに荷物がグチャグチャになるの、地味にストレスなんですよね…。

主なスペック

  • 型番: TB4-685（ブラック/タン/カーキ）
  • 使用サイズ: W60×D100×H80.5/69/55.5/44cm
  • 収納サイズ: 約W60×D50×H6.6cm
  • 重量: 約6.1kg（付属品含む）
  • 静止耐荷重: 天板30kg、ストレージ5kg
  • 材質: フレーム（アルミ合金）、天板・棚（MDF）
  • 価格: 約13,370円（税込）
あれ、6.1kgって結構重くないですか？
そこは正直なところデメリットだな。機能が充実している分、片手で持つのは厳しい。
参
「約6.1kgという重さがネックとなり、持ち運びの際に片手が塞がると感じる人もいます」
出典https://my-best.com
うーん、私には重そう…。
あと、天板がMDF素材だから焚き火の近くでは使えない。熱に弱いんだ。
MDF素材って何ですか？
木の繊維を圧縮して板にしたものだ。軽くて加工しやすいが、熱や湿気には要注意だな。水滴跡が残りやすいという声もある。
なるほど、焚き火の横には置けないんですね。

こんな人におすすめ

  • ファミリーキャンプでギアが多い人
  • キャンプサイトをすっきり見せたい人
  • 高さ調節で様々なスタイルに対応したい人
  • 車載時の荷物整理もしたい人

注意点

  • 約6.1kgとやや重い
  • MDF天板は焚き火の熱に弱い
  • 収納時の角が鋭利（怪我に注意）
  • 水滴跡が残ることがある

ソロキャンプのシステムキッチン「ソトメシンガーZ/RX」

Image Prompt: ソロキャンパーがソトメシンガーZにシングルバーナーを組み込んで調理している様子。コンパクトな調理スペース。

ソトメシンガーって名前、なんか強そうですね。
「外飯」を「シンガー」というのはDODらしいネーミングだな。これはソロキャンプ向けのゴトク内蔵型テーブルだ。
ゴトク内蔵？どういうことですか？
シングルバーナーをテーブルに組み込んで、ソロ用のシステムキッチンを作れるんだ。2台設置すれば2口コンロにもなる。
おお、ミニチュアのキッチンみたい！
ソトメシンガーZは高さ8段階、幅もスライド式で調整可能だ。手持ちのギアに合わせてカスタマイズできる。
参
「高さ8段階、幅がスライド式で調整可能な可変式ゴトクです。お手持ちのギアや状況に合わせて最適なサイズに調整できます」
出典https://dod.camp
焚き火でも使えるんですか？
スチール製だから、焚き火のゴトクとしても使える。直火調理派にはありがたい機能だな。

ソトメシンガーZ スペック

  • 使用サイズ: 約W52.5-78×D21×H8.5〜17cm（高さ8段階）
  • 収納サイズ: 約W23×D48×H4cm
  • 重量: 約2kg
  • 静止耐荷重: 10kg
  • 材質: スチール
RXっていうのもありますよね？何が違うんですか？
ソトメシンガーRXはZの進化系で、さらにコンパクトになっている。収納サイズが約17.5×17.5×4.5cmしかない。
小さっ！
ただし、コンパクトさを優先したためOD缶のシングルバーナーには対応していない。高さが合わないんだ。
参
「コンパクトさを優先したため、OD缶のシングルバーナーは高さが合わず使用できません」
出典https://dod.camp
え、じゃあCB缶専用ってことですか？
そうなる。自分のバーナーがOD缶なら、Zを選んだ方がいい。

こんな人におすすめ

  • ソロキャンプで調理環境を充実させたい人
  • シングルバーナーをテーブルに組み込みたい人
  • 焚き火調理もしたい人（Z）
  • とにかくコンパクトに持ち運びたい人（RX）

注意点

  • RXはOD缶バーナー非対応
  • RXは高さ調節不可
  • Zは収納サイズがやや大きめ

ワンポールテント派必見「ワンポールテントテーブル」

Image Prompt: ワンポールテントの中央ポールを囲むようにワンポールテントテーブルが設置されている様子。おしゃれなウッド調天板。

先生、ワンポールテント使ってるんですけど、真ん中のポール邪魔じゃないですか？
そう、あれは意外とデッドスペースになるんだよな。そこで活躍するのがワンポールテントテーブルだ。
テントの中にテーブル入れるんですか？
そうだ。このテーブルは半円形のテーブル2つで構成されていて、ポールを囲むように設置できる。磁石で簡単に接続できるから、近づけるだけでOKだ。
参
「ワンポールテントの中央に設置することで、通常は使いにくい中央部分をダイニングスペースやリビングとして活用できます」
出典https://dod.camp
へえ、分離もできるんですね。
そう。寝る時はテーブルを移動させてスペースを確保したり、半円だけ使ったりと柔軟に対応できる。
ちゃぶ台みたいで可愛い！ウッド調の天板もおしゃれですね。
デザイン性の高さは間違いなくDODらしいところだな。高さも3段階で調節可能だ。

主なスペック

  • 直径: 約90cm（組み立て時）
  • 高さ: 29.5cm/35cm/41.5cm（3段階）
  • 重量: 約4.7kg（付属品込み）
  • 静止耐荷重: 30kg
  • 対応ポール径: 最大32mm
  • 材質: 天板MDF、フレームアルミニウム
対応ポール径！これ確認しないとダメですね。
そうだ。購入前に自分のテントのポール直径は必ずチェックしてくれ。

こんな人におすすめ

  • ワンポールテントユーザー
  • テント内のデッドスペースを活用したい人
  • おしゃれなテント内空間を作りたい人

注意点

  • ポール直径32mmまで対応（要確認）
  • MDF天板は熱に弱い
  • ブラックカラーは販売終了

焚き火派の大本命「テキーラテーブルシリーズ」

Image Prompt: 焚き火を囲む囲炉裏スタイルのテキーラテーブル。鉄製プレートの上にダッチオーブンが置かれている。

先生、テキーラテーブルってお酒ですか？
はは、名前のインパクトはすごいよな。でもこれは焚き火で使えるテーブルなんだ。
焚き火！？テーブルが燃えちゃいません？
テキーラプレートは3mm厚の鉄製で、耐熱塗装が施されている。焚き火やBBQの炎が当たっても塗装が剥げにくい。ダッチオーブンを直接置いて調理することもできる。
参
「テキーラテーブルのプレートは3mm厚の鉄製で、耐熱塗装が施されているため、焚き火やBBQの炎が当たっても塗装が剥げにくく、焚き火の上での使用が可能です」
出典https://dod.camp
えっ、めちゃくちゃ便利じゃないですか！
しかもこのシリーズ、モジュール式で組み合わせが自由なんだ。プレートとレッグを別々に買って、自分好みのレイアウトを作れる。
LEGOみたい？
そんな感じだな。囲炉裏スタイルで焚き火台を囲んだり、2段ラックにしたり、ホームセンターのワンバイ木材と組み合わせたりもできる。
どんなパーツがあるんですか？
主なものを挙げると…
  • テキーラプレートM/S: 鉄製天板、耐熱塗装
  • テキーラウッドプレートM/C: 木製天板（ブナ材）
  • テキーラレッグS/M/L: テーブルの脚
  • テキーラローレッグ: お座敷スタイル用
  • テキーラキッチンレッグ: システムキッチン用
  • テキーラランタンレッグ: ランタンスタンド用
  • テキーラ180: コックピット型囲炉裏テーブル
うわ、多い…。初心者はどれから買えばいいんですか？
まずはプレート2枚とレッグ2つのセットから始めるといい。慣れてきたら追加していけばいい。
ワンバイ木材って何ですか？
ホームセンターで売っている規格サイズの木材だ。厚さ約19mmで、テキーラレッグにぴったり差し込める。これを使えば棚板を自作できるし、コストも抑えられる。
DIY好きにはたまらないですね。
ただし注意点もある。木製部品は火に直接当てないこと。それから、四角形の囲炉裏テーブルはバランス調整に時間がかかることがある。

こんな人におすすめ

  • 焚き火料理を楽しみたい人
  • 自分だけのレイアウトを作りたい人
  • DIYが好きな人
  • キャンプスタイルに合わせて拡張したい人

注意点

  • 木製部品は火の近くに置かない
  • 鉄製のためやや重量がある
  • 初期投資でパーツ代がかかる

天然木の高級感「キャナリーテーブル」

Image Prompt: 天然木天板のキャナリーテーブルがナチュラルなキャンプサイトに置かれている。朝の光が木目を美しく照らしている。

キャナリーって何ですか？カナリア？
「かなり良い」をもじったネーミングだ。DODには「ジミニーテーブル」という先行商品があって、「地味に良い」という意味だった。それを進化させたのがキャナリーだ。
参
「『地味に良い』と評された先行商品『ジミニーテーブル』をさらに進化させ、『かなり良い』テーブルとして開発されました」
出典https://dod.camp
へえ、ダジャレですね。でも天然木の天板、すごくおしゃれ！
ブナ材を使っていて、高級感がある。キャンプサイトの雰囲気を格上げしてくれるテーブルだな。
高さ調節もできるんですか？
3段階だ。21cm、42cm、56cmで、お座敷スタイルからミドルスタイルまで対応できる。ストレージメッシュも付属しているから、荷物置きにも便利だ。

キャナリーテーブルL スペック

  • 型番: TB8-807-WD
  • 使用サイズ: 約W120×D60×H21/42/56cm
  • 収納サイズ: 約W80×D10×H32cm
  • 重量: 約7.5kg（付属品含む）
  • 静止耐荷重: 10kg
  • 材質: 天板（天然木ブナ）、脚部（アルミ合金）
  • 価格: 約19,800円（税込）
Lサイズで4〜6人対応ですか。ファミリー向けですね。
Mサイズもある。幅が約94.5cmとひと回り小さくて、デュオキャンプやソロでゆったり使いたい人向けだな。
天然木ってお手入れ大変じゃないですか？
確かに、直射日光やカビには注意が必要だ。使用後は汚れを落として完全に乾燥させてから保管すること。それから、天然木だから個体差がある。木目や色合い、多少の反りや小傷は味わいだと思ってくれ。
え、新品なのに傷があることもあるんですか？
天然素材だからな。それが気になる人は、別のテーブルを選んだ方がいいかもしれない。

こんな人におすすめ

  • おしゃれなキャンプサイトを作りたい人
  • 天然木の温かみが好きな人
  • 高さ調節で様々なスタイルに対応したい人

注意点

  • 約7.5kgとやや重い
  • 天然木のため個体差あり
  • 直射日光・高湿度・カビに注意
  • 耐荷重10kgと控えめ

10秒設営！「マルチキッチンテーブル」

Image Prompt: マルチキッチンテーブルをアタッシュケースから開いて設営する様子。3段の棚に調理器具が整理されている。

マルチキッチンテーブルって、グッドラックテーブルと似てません？
確かに収納重視という点では似ているが、こちらは設営の速さが売りだ。アタッシュケース状態から天板を持ち上げるだけで、約10秒で3段棚が完成する。
参
「ロックを外し、天板を引き上げるだけで棚が完成し、約10秒で設営できる」
出典https://happycamper.jp
10秒！？ほんとですか？
ほぼワンタッチだな。収納容量も約170〜210リットルとグッドラックテーブル以上だ。背面がメッシュになっていて通気性がいいから、洗った食器を乾かすのにも使える。
棚の段数は変えられるんですか？
中央のジッパーを開放することで1段から3段まで切り替えられる。大型の荷物を入れたい時はジッパーを全開にすればいい。
あ、でもこれも結構重そうですね。約7.2kg…。
グッドラックテーブルより1kg以上重い。収納力の代償だな。それから高さ調節機能がないのも注意点だ。

主なスペック

  • 使用サイズ: 約W60×D48×H81cm
  • 収納サイズ: 約W60×D48×H7.5cm
  • 重量: 約7.2kg
  • 静止耐荷重: 30kg
  • 収納容量: 約170〜210L
  • カラー: ブラック、タン、カーキ

こんな人におすすめ

  • 設営・撤収を素早く済ませたい人
  • 調理道具やギアが多い人
  • キッチンテーブルとして使いたい人

注意点

  • 約7.2kgと重め
  • 高さ調節不可
  • MDF天板は熱に弱い

調理と食事を一体化「ダイキチテーブル」

Image Prompt: ダイキチテーブルにシングルバーナー2台を組み込んで、家族で鍋料理を囲んでいる様子。

ダイキチ…大吉？縁起良さそうですね。
「ダイニング」と「キッチン」を組み合わせて「ダイキチ」だ。つまり、調理と食事が同じテーブルでできるというコンセプトだな。
あ、なるほど！
このテーブルの特徴は、シングルバーナーを2台まで組み込めること。SOTO ST-310やイワタニのジュニアコンパクトバーナーなど、人気のバーナーに対応している。
参
「手持ちのバーナーで2バーナースタイルのキッチンを実現可能です」
出典https://dod.camp
ツーバーナーが買えなくても、手持ちのシングルバーナー2つで代用できるんですね！
そういうことだ。しかもバーナープレートの配置は自由に変えられる。中央に置いて鍋を囲んだり、端に寄せて調理スペースを確保したりできる。
ガス缶はどこに置くんですか？邪魔になりません？
テーブルの下や横にすっきり収納できる設計になっている。天板の上は広々と使える。
大きさはどのくらいですか？
使用サイズが約W145×D70cmで、4〜5人でゆったり使える。高さも3段階調整可能だ。
ファミリー向けですね。でもソロや2人だと大きすぎません？
そういう人には「チュウキチテーブル」という兄弟モデルがある。サイズが約W70×D70cmで、1〜2人向けだ。

ダイキチテーブル スペック

  • 使用サイズ: 約W145×D70×H25.5/35/56cm
  • 収納サイズ: 約W70×D20×H17cm
  • 重量: 約7.5kg
  • 推奨人数: 4〜5人

天然木テーブルのメンテナンス

先生、天然木テーブルって手入れ大変そうで不安なんですけど…。
基本的なケアを守れば、長く使えるぞ。

日々のお手入れ

  1. 柔らかい布で乾拭き
  2. 汚れは固く絞った布で水拭き後、すぐ乾拭き
  3. 化学雑巾、揮発性溶剤は使用しない

カビ対策

  1. 使用後は汚れを落として完全に乾燥してから保管
  2. 高温多湿、直射日光を避ける
  3. 換気と除湿を徹底
参
「使用後は汚れを落とし、完全に乾燥させてから保管することが指示されています。濡れたまま保管すると、カビや腐食、劣化の原因となります」
出典https://dod.camp
カビが生えちゃったらどうすればいいですか？
中性洗剤を薄めた水で拭き取るか、エタノールを使う。ただし塩素系漂白剤は絶対に使うな。木材を傷めてしまう。
